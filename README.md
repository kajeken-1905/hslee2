# My Vacation Plan

유럽 8개국 여행 체크리스트 · 도시 가이드 웹앱  
**사이트:** https://kajeken-1905.github.io/hslee2/

## 로컬 실행

```bash
npm install
npm run dev
```

## GitHub에서 사이트 보이게 하기

1. https://github.com/kajeken-1905/hslee2/settings/pages 접속
2. **Build and deployment → Source** = `Deploy from a branch`
3. Branch = `main` , folder = `/docs`
4. Save 후 1–2분 뒤 위 사이트 주소로 접속

코드는 `main` 브랜치, 배포용 빌드는 `docs/` 폴더에 있습니다.
